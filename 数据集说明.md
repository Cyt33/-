
# OpenVLA 数据集说明

## 1. 数据集概述

本项目使用 **LIBERO** 数据集进行训练和测试，这是一个用于机器人操作任务的大规模数据集。

| 属性 | 说明 |
|------|------|
| 数据集名称 | LIBERO (Long-Horizon Benchmark for Robot Manipulation) |
| 样本数量 | 约 50,000 条轨迹 |
| 任务类型 | 10 个不同的操作任务 |
| 数据格式 | RLDS (Robotic Learning Data Set) |

## 4. 数据预处理

### 4.1 预处理步骤

```bash
# 运行预处理脚本
python preprocess_libero.py
```

### 4.2 预处理配置

```python
# preprocess_libero.py 配置说明
DATA_ROOT = "./data/modified_libero_rlds"    # 原始数据路径
OUTPUT_DIR = "./data/processed_libero"       # 输出路径

# 数据集划分比例
TRAIN_RATIO = 0.7    # 训练集 70%
VAL_RATIO = 0.2      # 验证集 20%
TEST_RATIO = 0.1     # 测试集 10%

# 图像预处理参数
IMAGE_SIZE = (224, 224)
NORMALIZE_MEAN = [0.485, 0.456, 0.406]
NORMALIZE_STD = [0.229, 0.224, 0.225]
```

### 4.3 数据清洗规则

| 清洗规则 | 说明 |
|----------|------|
| 无效图像过滤 | 移除模糊、损坏或空白的图像 |
| 异常动作过滤 | 移除超出合理范围的动作值 |
| 短轨迹过滤 | 移除长度小于 10 帧的轨迹 |
| 重复样本过滤 | 移除完全重复的轨迹 |

## 5. 数据集划分

### 5.1 划分策略

采用分层划分策略，确保每个任务类型在训练集、验证集和测试集中都有足够的样本：

```python
# 分层划分代码逻辑
from sklearn.model_selection import train_test_split

# 按任务类型分层
train_samples, temp_samples = train_test_split(
    all_samples, 
    test_size=0.3, 
    stratify=task_labels,  # 按任务类型分层
    random_state=42
)

val_samples, test_samples = train_test_split(
    temp_samples, 
    test_size=1/3, 
    stratify=temp_task_labels,
    random_state=42
)
```

### 5.2 划分结果

| 数据集 | 样本数量 | 占比 |
|--------|----------|------|
| 训练集 | ~35,000 | 70% |
| 验证集 | ~10,000 | 20% |
| 测试集 | ~5,000 | 10% |

## 7. 数据加载

### 7.1 使用 data_loader.py

```python
from data_loader import load_trajectory

# 加载单条轨迹
images, actions, instruction = load_trajectory(
    subset="libero_goal", 
    traj_id="000001"
)

# images: 图像序列 (list of numpy arrays)
# actions: 动作序列 (numpy array)
# instruction: 指令文本 (string)
```

### 7.2 批量加载

```python
from torch.utils.data import DataLoader
from dataset import LIBERODataset

dataset = LIBERODataset(data_dir="./data/processed_libero/train")
dataloader = DataLoader(dataset, batch_size=16, shuffle=True)

for batch in dataloader:
    images = batch["images"]      # (B, T, C, H, W)
    actions = batch["actions"]    # (B, T, 7)
    instructions = batch["instructions"]  # (B,)
```

## 9. 数据集使用注意事项

1. **数据版权**: LIBERO 数据集仅供研究使用
2. **数据完整性**: 下载后请校验文件完整性
3. **存储要求**: 原始数据约 50GB，预处理后约 80GB
4. **加载效率**: 建议使用 PyTorch DataLoader 进行批量加载
5. **数据增强**: 训练时建议使用随机裁剪、翻转等增强策略
