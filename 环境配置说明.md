
# OpenVLA 环境配置说明

## 1. 环境要求

### 硬件要求
- **GPU**: NVIDIA GPU with CUDA support (建议 >= 24GB VRAM)
- **内存**: >= 32GB RAM
- **存储**: >= 100GB 可用空间

### 软件要求
- **操作系统**: Ubuntu 20.04+/Windows 10+/macOS 12+
- **Python**: 3.10.x
- **PyTorch**: 2.1.0+
- **CUDA**: 11.8+

---

## 2. 环境搭建步骤

### 2.1 安装 Anaconda

```bash
# 下载并安装 Anaconda
# 官网: https://www.anaconda.com/products/distribution

# 创建虚拟环境
conda create -n openvla python=3.10 -y
conda activate openvla
```

### 2.2 安装 PyTorch

```bash
# 安装 PyTorch 2.1.0 with CUDA 11.8
pip3 install torch==2.1.0 torchvision==0.16.0 torchaudio==2.1.0 --index-url https://download.pytorch.org/whl/cu118
```

### 2.3 安装项目依赖

```bash
# 安装基础依赖
pip install -r requirements.txt

# 安装额外依赖
pip install peft==0.11.1 accelerate==0.25.0 transformers==4.37.2
pip install timm==0.9.12 torchvision==0.16.0
pip install tensorflow==2.15.0  # 用于读取 RLDS 数据集
pip install scikit-learn matplotlib tqdm
```

### 2.4 配置国内镜像（可选）

```bash
# 设置 HuggingFace 国内镜像
export HF_ENDPOINT=https://hf-mirror.com

# 设置 PyPI 国内镜像
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

## 4. 常见问题与解决方案

### 问题1: CUDA 版本不兼容
```
RuntimeError: CUDA error: invalid device function
```
**解决方案**: 确保 CUDA 版本与 PyTorch 版本匹配，建议使用 CUDA 11.8

### 问题2: 内存不足
```
RuntimeError: CUDA out of memory
```
**解决方案**: 使用 INT8/INT4 量化加载模型，或减小 batch_size

### 问题3: 依赖版本冲突
```
ImportError: cannot import name 'xxx' from 'xxx'
```
**解决方案**: 使用 `pip install --force-reinstall` 重新安装指定包
